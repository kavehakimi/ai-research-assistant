# AI Research Assistant

An AI-powered research planning and idea development tool built using Google AI Studio and Gemini.

## Overview

AI Research Assistant helps researchers and PhD students rapidly generate structured research plans from a research topic or research question.

The application can assist with early-stage research design, proposal development, and literature review planning.

## Features

* Research problem generation
* Research objectives generation
* Research questions generation
* Research methodology suggestions
* Literature review outline generation
* Research keyword recommendations
* Future research direction suggestions

## Technologies

* Google AI Studio
* Gemini
* Cloud Run
* TypeScript
* Web Application Architecture

## Example Use Cases

* PhD proposal preparation
* Research idea exploration
* Literature review planning
* Academic project design

## Deployment

The application was deployed using Google Cloud Run.

## Project Background

This project was developed during Google's 5-Day AI Agents Intensive Course as an exploration of AI-powered research support systems and agent-based application development.
